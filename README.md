## 📌 Project Overview

**FindMyRoomie** is a full-stack web application that helps users **find roommates** and **post about room availability**. The platform makes it easier for students, professionals, and renters to connect with like-minded individuals looking for a place to stay. It features listing search, posting management, Firebase authentication, and responsive UI.

---

## 🚀 Key Features

- 🔐 **Authentication**
  - Firebase Google Sign-in
- 🏘️ **Roommate Listings**
  - Browse roommate posts with key details
- ✏️ **Post Room Availability**
  - Authenticated users can create, update, and delete their posts
- 👥 **My Posts**
  - Personalized dashboard for managing listings
- 📱 **Responsive Design**
  - Optimized for mobile and desktop views
- 💬 **User Notifications**
  - Toast and alert messages for actions
- 🌐 **Live Deployment**
  - Firebase Hosting (Spark Plan)
